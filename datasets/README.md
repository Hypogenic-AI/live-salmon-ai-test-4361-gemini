# Datasets: Live Salmon Test for AI Scientists

This research project uses **interactive simulators** rather than static datasets as its primary data-generating processes ("Live Salmon").

## Simulators
Simulators are contained within the `code/` directory repositories:
- **NewtonBench Tasks**: 324 scientific law discovery tasks.
- **PhysGym Problems**: 97 physics discovery problems.
- **GravityBench Simulations**: Binary star system orbital dynamics.

## Auxiliary Data
- **Pitfall Logs**: Trace logs and audit data for detecting common AI Scientist failure modes. Located in `code/AIScientistPitfalls/`.

## Download Instructions
Since most "data" is generated on-the-fly by the benchmark simulators, no massive static downloads are required beyond the code repositories.
