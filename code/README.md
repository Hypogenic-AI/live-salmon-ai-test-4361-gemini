# Code Repositories: Live Salmon Test for AI Scientists

This directory contains the implementations of AI Scientist frameworks and benchmarks.

1. [AI-Scientist](AI-Scientist/)
   - The original Sakana AI implementation.
2. [AI-Scientist-v2](AI-Scientist-v2/)
   - The upgraded template-free agentic tree search version.
3. [NewtonBench](NewtonBench/)
   - Physics law discovery benchmark with law shifts.
4. [GravityBench](GravityBench/)
   - Strategic orbital dynamics discovery benchmark.
5. [PhysGym](PhysGym/)
   - Interactive physics discovery benchmark with prior control.
6. [AIScientistPitfalls](AIScientistPitfalls/)
   - Tools and data for auditing AI Scientist trace logs.

## Setup
Each repository has its own `README.md` and dependency requirements. It is recommended to use the project's root `.venv` and add specific dependencies as needed.
