4. [Gravity-Bench-v1](2501.06789_GravityBench.pdf)

5. [PhysGym](2507.15550_PhysGym.pdf)
   - Benchmark for interactive physics discovery with controlled priors.
6. [Hidden Pitfalls](2509.08713_Hidden_Pitfalls.pdf)
   - Critical analysis of failure modes in AI Scientist systems.
